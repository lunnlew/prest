# 图片导出功能（PDF/HTML/WebP）

## 概述

通用图片导出功能，将 Markdown 预览结果导出为图片格式，与"小红书排版出图"区别在于：
- 导出为标准预览样式的截图
- 无小红书专属装饰元素
- 适用于通用文档分享场景

## 功能清单

| ID | 功能 | 优先级 | 状态 | 备注 |
|----|------|--------|------|------|
| IMG-001 | 导出 PNG 图片 | P1 | ⏳ | 将完整预览区渲染为 PNG |
| IMG-002 | 导出 PDF | P1 | ⏳ | 多页 PDF，适合打印分享 |
| IMG-003 | 导出 HTML | P1 | ⏳ | 将预览结果导出为独立 HTML 文件 |
| IMG-004 | 导出 WebP | P2 | ⏳ | 更小文件体积，适合网络传输 |
| IMG-005 | 批量导出多个文件 | P2 | ⏳ | 批量导出/下载多个文档（zip） |
| IMG-006 | 导出范围选择 | P2 | ⏳ | 全篇/选定段落/自定义页码 |

## 技术实现规划

方案 | 说明 | 适用格式 | 备注
-----|------|---------|-----
html2canvas | DOM → Canvas → 图片 | PNG/WebP | 最成熟，兼容性好
dom-to-image-more | SVG foreignObject 方案 | PNG/WebP | 体积更小
window.print() / @media print | 原生浏览器打印 | PDF | 零依赖
Puppeteer (服务端) | 高质量渲染导出 | PDF/PNG | 需要后端，未来考虑
