# 小红书排版出图

## 概述

将 Markdown 内容一键转换为小红书风格的竖版图片，支持模板、配色、字体自定义，适用于创作者快速制作小红书图文内容。

## 15.6 技术实现规划

| 技术方案 | 说明 | 备选 |
|----------|------|------|
| html2canvas | 将 DOM 渲染为 Canvas 再导出 PNG | 最成熟，兼容性最好 |
| dom-to-image-more | 纯 SVG foreignObject 方案 | 体积更小，但对复杂 CSS 支持有限 |
| Canvas 2D 自绘 | 手动绘制所有元素 | 最大灵活度但开发量大 |
| Puppeteer SSR | 服务端渲染（未来） | 更高质量但需后端 |

**推荐方案:** 前端 html2canvas 或 dom-to-image-more + 自定义 React 卡片组件

## 15.7 推荐实现流程

```
Markdown 输入
    ↓
解析为 Markdown AST
    ↓
映射为小红书卡片组件树 (XHSCardRenderer.tsx)
    ↓
应用预设样式/模板 (XHSStyles.ts)
    ↓
渲染为独立不可见 div
    ↓
html2canvas 捕获 → Canvas → PNG 下载
```
